U.S. Congressional World View

Every month, the U.S. Congress names countries in bills, hearings, and floor speeches. The flags below show which nation led the count each month, a proxy for where American foreign-policy attention lands. Inspired by The Pudding’s landmark analysis of the New York Times, this project applies a similar lens to the legislative record.

Each flag represents the most-mentioned country in congressional records that month. Tap a cell for details.
